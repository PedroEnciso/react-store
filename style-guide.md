# Front-end Style Guide

## Layout

The designs were created to the following widths:

- Mobile: 375px
- Desktop: 1440px

## Colors

### Primary

- Purple: 4D48AF

### Secondary

- Pale Purple: hsl(243, 42%, 48%, 10%)

### Neutral

- Very Light Grey: F8F8FA

## Typography

- Dark Text: 21212D
- Light Text: 2C2C38

### Body Copy

- Font size: 18px

### Fonts

- Family: [Libre Franklin](https://fonts.google.com/specimen/Libre+Franklin)
- Weights: 300, 600, 700
